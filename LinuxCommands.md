Basic File Navigation
---------------------
- `cd`
- `cp`
- `mv`
- `rm`
- `mkdir`
- `rmdir`
- `ls`
- `pwd`

File reading
------------
- `cat`
- `head`
- `tail`
- `less`
- `grep [options] pattern [file]`

File transfer
-------------
- `scp [-P <port>] <source_user>@<source_domain>:<source_path> <destination_user>@<destination_domain>:<destination_path>`
